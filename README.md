# Police Station Management System

A robust Java-based desktop application designed to streamline law enforcement operations, manage criminal records, and track officer assignments.

## 🚀 Overview
This project was developed to digitize the manual record-keeping processes in a police station. It provides a centralized database for managing cases, suspects, and administrative tasks, ensuring data integrity and quick retrieval.

## ✨ Key Features
* **Case Management:** Create, update, and track the status of criminal cases.
* **Officer Directory:** Maintain detailed records of personnel, including rank and department.
* **Criminal Records:** Store and search for suspect profiles and criminal history.
* **Search Functionality:** Filter records by ID, name, or case type for rapid information access.
* **Secure Login:** Role-based access to ensure only authorized personnel can modify data.

## 🛠️ Tech Stack
* **Language:** Java
* **GUI Framework:** AWT
* **Database:** MySQL
* **Environment:** Developed and tested on Linux and Window

## 📥 Installation & Setup
### Windows
1. Open the project in your IDE (VS Code/IntelliJ/Eclipse).
2. Ensure the JDK path is set in your Environment Variables.
3. Run `Main.java`.

### Linux (Kali/Ubuntu)
1. Install OpenJDK: `sudo apt install default-jdk`
2. Compile: `javac Main.java`
3. Run: `java Main`
